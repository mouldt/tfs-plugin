The versionspec is the version of the source you want to retrieve. You can specify a version by:

  • Date/time (D10/20/2005)
  • Changeset version (C1256)
  • Label (Lmylabel)
  • Latest version (T)
  • Workspace version (Wworkspacename;owner)
If no versionspec is provided, the latest server version of the source is retrieved into your workspace.
